Аренда автомобилей и велосипедов
Автоматизированная система с использованием Typescript, Node.JS и React.
АС предназначена для повышения оперативности и качества оказания услуг по аренде автомобилей и велосипедов Заказчиком, создание нового канала продаж в интернете. Основным назначением АС является автоматизация процесса приобретения услуг организации Заказчика.
Для реализации поставленных целей система решает следующие задачи:
1. Поиск автомобилей по их модели, местонахождению и желаемым датам.
2. Заказ автомобиля: регистрация/авторизация пользователей в системе, создание нового бронирования автомобиля, скрытие автомобиля от повторных заказов на занятые даты.
3. Предоставление административного интерфейса для просмотра, создания, редактирования и удаления моделей, автомобилей, локаций, заказов, пользователей.

В рамках проектирования системы на основе результатов анализа предметной области и конкурентов было принято решение составить систему из следующих компонентов:
  1. Веб-сайт на React.
  2. Админ-панель на React.
  3. Бэкенд на Node.JS и PostgreSQL.
Процессы
Основной процесс - бронирование автомобиля. Соответственно, всё содержимое сайта либо реализовывает этот процесс, либо подталкивает пользователя к покупке.
  1. Заполните форму
2. Выберите автомобиль
3. Заполните данные о себе
4. Дождитесь подтверждения заказа менеджером
5. Распечатайте данные о бронировании для дилерского центра
Преимущества
Расширяемая система из трёх программных компонентов.
Современные технологии. В проекте используются React, Node.JS и PostgreSQL.
Простой процесс заказа автомобиля. За каждым программным решением лежит анализ предметной области и конкурентов
Панель администрирования. Менеджер может легко отредактировать всю информацию в системе из единой панели.
RESTful API. Легко интегрируйте систему с бухгалтерией, CRM или складским ПО.
